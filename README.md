# Shoreline Change Detection of the Kwa River (2010–2025)

This project analyzes shoreline changes along the Kwa River in Nigeria using historical satellite imagery and GIS techniques.

The study uses Google Earth Pro imagery and QGIS to identify shoreline movement, analyze environmental changes, and produce spatial outputs useful for coastal monitoring and land management.

## Objectives

• Detect shoreline changes between 2010 and 2025  
• Digitize shoreline positions using satellite imagery  
• Analyze shoreline movement patterns  
• Produce GIS maps showing shoreline evolution  

## Study Area

Kwa River, Cross River State, Nigeria.

## Data Sources

- Google Earth Pro historical imagery
- Digitized shoreline vectors
- GIS spatial datasets

## Tools Used

- QGIS
- Google Earth Pro
- Python (for geospatial analysis where applicable)

## Methodology

1. Acquire historical satellite imagery
2. Digitize shoreline positions for different years
3. Overlay shoreline layers in QGIS
4. Analyze spatial changes
5. Generate maps and spatial outputs

## Repository Structure

data/ → spatial datasets and digitized shoreline layers  
maps/ → final cartographic outputs
